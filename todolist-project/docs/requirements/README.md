## 项目概览

### 项目名称
TodoList 待办事项管理系统

### 项目目标
- 演示完整的 SDLC 开发流程
- 提供一个标准化的 Spring Boot 示例项目
- 为团队提供可复用的项目模板

### 技术栈
- **后端框架**: Spring Boot 3.2.0
- **编程语言**: Java 17
- **数据库**: MySQL 8.0+
- **数据库迁移**: Flyway
- **安全框架**: Spring Security + JWT
- **API 文档**: Swagger/OpenAPI

### 项目范围

#### 包含的功能
- ✅ 用户注册和登录 (基于 JWT)
- ✅ 待办事项 CRUD 操作
- ✅ 待办事项状态管理 (待处理/已完成)
- ✅ 分类/标签管理
- ✅ 搜索和过滤功能
- ✅ 个人信息管理

#### 不包含的功能
- ❌ 实时通知
- ❌ 社交分享
- ❌ 高级报表
- ❌ 移动端原生应用
- ❌ 多租户支持

## 非功能需求

### 性能要求
- API 响应时间: ≤ 500ms (95分位)
- 数据库查询: ≤ 100ms (95分位)
- 并发用户: ≥ 50 用户

### 安全要求
- 基于 JWT 的身份认证
- 密码 BCrypt 加密
- 登录失败次数限制 (5 次)
- SQL 注入防护
- XSS 防护
- 数据隔离 (用户只能访问自己的数据)

### 质量要求
- 单元测试覆盖率: ≥ 80%
- 代码符合阿里巴巴 Java 开发规范
- API 文档完整性: 100%
- 代码注释率: ≥ 30% (核心类)
