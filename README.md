# Claude Code + Happy + glm4.7 实战指南

> Windows + Java Spring Boot + Ruoyi 快速上手

## 简介

本指南面向有 Java 开发经验的开发者，介绍如何：
- 在 Windows 上搭建 Claude Code + glm4.7 开发环境
- 配置 Happy 实现手机远程开发
- 开发自定义 Skills 应用于实际项目
- 在 Java Spring Boot + Ruoyi 项目中实践
